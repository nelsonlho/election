## 步驟

1. **讀起法諸章**（體卦、用卦、變卦、起卦總法）——先求**算法**，
   得算法則六十×六十對照表僅作驗證，不須逐格錄。
2. **推導干支→卦之映射**——男命卦、女命卦各自起例
   （對照表已示：女命甲子〜壬申一組皆配艮宮山某卦，即女命定上卦、男命定下卦之屬）。
3. **錄卦義吉凶**——六十四卦逐卦婚配斷語（吉／凶／平＋斷語原文）。
4. **實作 `lib/hehun.ts`**：
   - `mingGua(gan, zhi, sex) → 八卦`
   - `heGua(女命, 男命) → { 體卦, 用卦, 變卦, verdict }`
5. **對勘驗證**——自對照表（書 136-138）抽樣十二格以上，
   算法所得與原書全合方可用（法同陰陽不將之驗）。
6. **UI**——新頁籤「合婚」：男女生年二欄 → 卦象、斷語、
   並附日課合參之連結（帶入嫁娶尋吉日）。
7. **Commit** 分二：engine＋驗證一筆，UI 一筆。

## 風險

- 起法章若為敘例而無明式，須自對照表反推（如陰陽不將故事）。
- 卦義斷語卷帙大（六十四卦），酌情先錄婚配所用之宮（對照表所見以艮宮等為主），
  餘宮續補。
- 掃描字糊之格，闕疑勿妄錄，記於本檔「存疑」節。

## 全章讀畢（書 146-153）——「變卦／逐卦斷語」乃虛列

重讀第二期卦例全章（PDF 105-107，2000×1500 字清）定案：

- **原書無獨立變卦**：章題「變用二卦起法」＝用卦起法（已錄）；書 149 註明斷法
  純以體用二卦六親生剋，動爻（用爻）之法乃「用動爻生體卦」之救法，概念已涵（用生體吉）。
- **無六十四卦逐卦斷語表**：書 149-152 例一〜五皆京房六親納甲斷
  （體用五行生剋＋官子二爻受病＋日辰吊合），非固定卦辭。斷法即算法，已全錄。
- 故 UNENCODED 類五原列「變卦、爻變、卦定逐卦斷語」係前誤——源本無此定式。
  合婚斷法**方法完備**，非殘章。
