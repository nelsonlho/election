# 剋擇擇日 UI 綱領

> 要旨：**表單量恆定，選項深度藏於摺疊與分科。**

## 一、漸進揭示（Progressive Disclosure）

首屏查詢表單恆只三類欄位，不因功能增而膨脹：

| 欄位 | 說明 |
|------|------|
| 事類 | 欲擇之事（嫁娶、安牀、出行……） |
| 日期 | 起始日＋尋日範圍（單日查則為查詢日） |
| 命造 | 隨事類而現之生年欄 |

其餘一切選項，一律藏於「進階選項」摺疊區。

## 二、進階摺疊

- 以 `<details>` 摺疊區收納非必要選項：時辰、門派取捨（神煞寬嚴）等。
- 預設收合；諸選項須有合理預設值，客不展亦可用。
- 新增選項時：先問「九成客戶須改此值否？」——否，則入進階區。

## 三、事類分科

- 事類以**二層籤選**呈現：上列科籤（tab 式底線），下列該科事籤（膠囊式）。
  廿餘事盡列而恆佔二行，任一事點二下可達，選中恆顯——優於下拉之藏。
- 分科如下：
  - **婚嫁**：嫁娶、納采、安牀
  - **居家**：入宅、動土、修造
  - **出行營商**：出行、開市
  - **造葬**：安葬、破土
- 首層科目以四至五為限；新事類入既有科，不另增層。
- 事類定義集中於 `lib/events.ts` 之 `EVENT_CATEGORIES`，UI 自動生成，不手改選單。

## 四、隨事現欄

- 命造欄隨事類而變：嫁娶現「女命生年」（必填），餘事現「本命生年」（可留空）。
- 新事類須於定義中聲明所需命造欄，UI 據以自動顯隱。
- 同時最多現一至二命造欄；逾此則移入進階區。

## 五、偏好記存

- 生年、常用事類存於 `localStorage`，再訪自動代入。
- 鍵名統一前綴 `kezhai:`（如 `kezhai:femaleYear`）。
- 只存客戶輸入，不存查詢結果。

## 六、結果過濾後置

- 顯隱濾器（兼列平日、神煞明細等）置於**結果區**頂部，不入查詢表單。
- 濾器即時生效，不須重查。

## 五之二、時間跨度（過去未來皆可查）

- 起始日**可設任意日**（含過去，無 min 限）；`type="date"` 自由選。標「可設過去」以曉客。
- 二端各有 preset 列，語義對稱、可組任意窗：
  - **往前**（以起始日為窗末、退 N 日）：卅／九十／半年／一年——窗＝起始前 N 日 至 起始日。
  - **往後**（以起始日為窗首、進 N 日）：卅／九十／半年／一年／三年——窗＝起始日 至 起始後 N 日。
  - 二者皆以**起始日**為樞：往前向左（起始為末）、往後向右（起始為首）。
- 單查上限 `MAX_SPAN_DAYS`（約三年／1100 日，engine 常數）；~1100 日全評約 0.7 秒，尚可。
  逾此須移窗分查。熱圖月分頁（六之二）承多月無虞（三年＝36 月頁）。

## 六之二、月曆熱圖（尋吉日結果全景）

- 尋吉日結果頂置**月曆熱圖**：全範圍逐日按評等染色（吉硃紅・平灰・凶玄黑），
  週一為首之月格，範圍外之日淡出。使客一目全局——何處吉、何處密、何處空。
- 熱圖**點日即展課詳**（`CalendarHeatmap` → `selKey` → `DayCard defaultOpen`），
  含濾器所不列之凶日亦可點究其因。補「線性卡片只見吉日」之不足。
- 熱圖為總覽，不取代下方吉日卡片列；二者並存。

## 六之三、分享固定連結

- 三頁查詢皆可**複製連結**：query 編入網址參數（`?tab=…&ev=…&f=…&s=…&d=…&mt=…&off=…`），
  開連結即代入諸欄還原此查——課果可傳家人、擇日師，可書籤，可跨機重現。
- 尋吉日連結**自動尋日**（`runRef` 於 mount 後觸發）；單日查、合婚即時計算，代欄即現。
- 鍵名精簡（`f`女命·`b`本命·`s`起日·`d`日數·`mt`座山·`jx`兼向·`xm`仙命·`fm`/`bm`生月·`cat`科·`m`男命）。
- 只編查詢輸入，不編結果；`localStorage` 存個人常用值，網址參數則供分享——二者不衝，網址參數優先代入。

## 六之四、列印課單

- 尋吉日擇定之日（熱圖點選者）可**列印課單**：`CourseSheet` 呈事類、命造、公農干支、
  評定、吉凶逐條詳批、吉時、沖／胎神／彭祖——擇日師交客之單。
- 印術：`CourseSheet` 恆在 DOM，螢幕以 inline `display:none` 隱（Tailwind v4 會略去無類名之裸 CSS 規則，
  故不以 class 隱）；`@media print` 內 `display:block !important` 覆 inline 而顯，並 `visibility` 隱其餘，白底黑字省墨。
- 「列印課單」鈕呼 `window.print()`。

## 七、色制

| 物 | 色 | 義 |
|----|----|----|
| 吉（評等） | 硃紅底白字 | 吉日 |
| 凶（評等） | 玄黑底 | 凶日 |
| 平（評等） | 灰底 | 平日 |
| 建除黃道／可用 | 金底（amber） | 吉神 |
| 建除黑道／勿用 | 玄底 | 凶神 |
| 通書宜（詞條） | 硃紅籤 | — |
| 通書忌（詞條) | 玄墨籤 | — |

- 全 app 一律，不得一處金一處紅。
- 明暗雙主題皆須可讀（`dark:` 變體必備）。

## 八、文字

- 全 UI 正體中文（臺灣慣用），擇日慣例字從古（牀不作床）。
- 曆法庫輸出一律經 `toTraditional()`（OpenCC + 校字表）轉換後方可顯示。
